Backup Instructions

The backup of your hass.io snapshots is performed by executing a REST service hosted by this add-on at the host port configured in the details for this add-on. There are no arguments needed to be passed to the service request. Everything needed is configured in the add-on configs. You simply need to send a GET operation to this address:

{{ request.scheme }}://{{ request.get_host }}{% url 'gb:doBackup' %}

You can use Home Assistant's RESTful Command to integrate this add-on's REST service into your own scripts and automations. You may want to use a REST testing tool like Postman to perform initial testing.
